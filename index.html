Common Cyber Threats

Understanding attack types is the first step in defending against them.

🎣

Phishing

Deceptive emails, messages, or sites that trick users into revealing credentials or downloading malware. The #1 attack vector for data breaches worldwide.

CRITICAL
🔒

Ransomware

Malware that encrypts your files and demands payment for decryption. Attacks have surged 13% year-over-year, targeting hospitals, schools, and businesses.

CRITICAL
🕵️

Social Engineering

Manipulating people psychologically to divulge confidential information. Bypasses technical defenses entirely by targeting the human layer.

HIGH
💉

SQL Injection

Inserting malicious SQL code into input fields to manipulate or extract database contents. One of the oldest and most prevalent web vulnerabilities.

HIGH
🌊

DDoS Attacks

Flooding a server with traffic from thousands of sources to knock it offline. Can take down websites, APIs, and entire infrastructures within minutes.

HIGH
👤

Man-in-the-Middle

Secretly intercepting and potentially altering communications between two parties. Common on unsecured Wi-Fi networks and unencrypted connections.

MEDIUM

Security Best Practices

Foundational habits that dramatically reduce your attack surface.

01

Use Strong, Unique Passwords

Each account should have a unique password of 16+ characters. Use a password manager to generate and store them securely.

02

Enable Multi-Factor Authentication

MFA stops 99.9% of automated credential attacks. Enable it on every account that supports it, prioritising email and banking.

03

Keep Software Updated

Unpatched software is the #1 exploit vector. Enable automatic updates for your OS, browser, and applications.

04

Verify Before You Click

Hover over links to inspect the real URL. Verify sender addresses. When in doubt, go directly to the official website.

05

Back Up Your Data

Follow the 3-2-1 rule: 3 copies, on 2 different media, with 1 stored offsite or in the cloud. Your best ransomware defense.

Security Tools to Know

Industry-standard tools used by security professionals and ethical hackers.

NETWORK

Nmap

The gold standard for network discovery and port scanning. Maps open ports and services across hosts.

ANALYSIS

Wireshark

Captures and analyses live network traffic. Essential for spotting anomalies and debugging protocols.

PENTESTING

Metasploit

Framework for developing and executing exploits. Used by pen testers to validate vulnerabilities legally.

WEB

Burp Suite

Intercepts and modifies HTTP/S traffic. Industry-standard for web application security testing.

PASSWORD

Hashcat

World's fastest password recovery tool. Used to audit password strength via GPU-accelerated cracking.

DISTRO

Kali Linux

The go-to OS for penetration testing, pre-loaded with 600+ security and forensics tools.
